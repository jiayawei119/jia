java的基础知识

1 .java中字符创无‘/0’由于此字符的作用主要是为了方便计算字符的长度，而java的String中有自带的length来计算字符串的长度。因此无需在最后包含'/0'.

2 .java属于编译与解释共存的语言。由集成开发环境进行编译最后使用JVM虚拟机将字节码文件。class文件解释为可以执行的计算码，可以一次编译，多次执行但不可跨操作系统
